# Vue Logic Calculator

## Projectbeschrijving
Dit project is een eenvoudige calculator gebouwd met Vue.js. Het project is gemaakt om te oefenen met frontend development en om beter te leren werken met componenten, data en basislogica in Vue.

## Doel
Het doel van dit project is om mijn kennis van Vue.js te verbeteren en te leren hoe je een interactieve webapplicatie kunt bouwen met componenten en reactieve data.

## Functionaliteiten
- Basis rekenfuncties (optellen, aftrekken, vermenigvuldigen en delen)
- Simpele en overzichtelijke gebruikersinterface
- Mogelijkheid om twee getallen in te voeren
- Resultaat wordt direct weergegeven
- Reset functie

## Gebruikte technologieën
- Vue.js
- JavaScript
- HTML
- CSS
